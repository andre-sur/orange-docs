Description du projet¶

Ce projet a pour objectif de développer une application web de gestion immobilière, permettant la gestion des profils et des locations.

L'application offre une interface simple et performante pour les utilisateurs.

Les principales fonctionnalités sont :

  • Gestion des profils utilisateurs

  • Gestion des biens à louer (lettings)

  • Interface d'administration sécurisée
