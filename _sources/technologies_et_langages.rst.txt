Technologies et langages utilisés
=================================

Ce projet utilise un ensemble de technologies modernes pour le développement, le déploiement et la documentation.

Langages de programmation
-------------------------

- **Python** : Langage principal utilisé pour le backend, via le framework Django.
- **HTML/CSS** : Utilisés pour les templates et le rendu des pages.
- **reStructuredText (reST)** : Utilisé pour la rédaction de la documentation avec Sphinx.

Frameworks et bibliothèques
---------------------------

- **Django** : Framework web complet pour le développement rapide de l'application.
- **Gunicorn** : Serveur WSGI utilisé pour le déploiement en production.
- **Whitenoise** : Sert les fichiers statiques efficacement en production.
- **Sphinx** : Génère la documentation technique du projet.
- **sphinx_rtd_theme** : Thème utilisé pour la publication sur Read the Docs.

Conteneurisation et déploiement
-------------------------------

- **Docker** : Utilisé pour créer une image isolée du projet, facilitant le déploiement.
- **Render** : Plateforme d’hébergement utilisée pour mettre l’application en ligne.

Contrôle de version
-------------------

- **Git & GitHub** : Utilisés pour le versionnage du code source et l’hébergement du dépôt distant.
