Guide d’utilisation¶

Ce guide décrit comment utiliser l’application Orange County Lettings du point de vue d’un utilisateur final. Il présente les principales fonctionnalités présentes sur l’interface web.

Accès à l'application

L'application est accessible via l’URL suivante :

https://orange-vpow.onrender.com/

Navigation générale

La page d’accueil propose deux sections principales :

  • Lettings : Locations disponibles

  • Profiles : Profils des utilisateurs

Menu de navigation situé en haut à droite.

Cas d’utilisation

### 1. Consulter la liste des locations

  • Depuis la page d’accueil, cliquer sur “Lettings”.

  • Une liste des logements disponibles s’affiche.

### 2. Consulter les détails d’un logement

  • Depuis la liste des locations, cliquer sur une location.

  • Les informations suivantes sont affichées : - Titre - Adresse complète - Code postal, ville, état - Pays

### 3. Consulter les profils utilisateurs

  • Cliquer sur “Profiles” depuis le menu.

  • Une liste des utlisateurs s’affiche.

### 4. Voir le détail d’un profil

  • Cliquer sur un nom d’utilisateur.

  • La page affiche : - Nom d’utilisateur - Ville préférée

Public cible

L’application permet : - accès à des données de location (agent immobilier) - Modification de la base via /admin (pour administrateurs autorisés) - Base de profils et préférence

Limitations actuelles

  • Création et modification uniquement via /admin (pas d'interface directe publique).

  • Site sans portail d'accès de session (hors admin).
